# 📊 Sales Performance Dashboard

## Overview
The **Sales Performance Dashboard** is an interactive Power BI dashboard that provides insights into sales, profit, production costs, discounts, and product performance. It enables users to analyze business performance using dynamic filters for **Category**, **Month**, and **Year**.

---

## Features

- Interactive slicers for:
  - Category
  - Month
  - Year
- KPI Cards:
  - Total Orders
  - Total Sales
  - Total Production Cost
  - Total Profit
  - Total Discount
- Monthly Profit & Discount Analysis
- Yearly Profit & Discount Comparison
- Production Cost Analysis by Year
- Profit Analysis by Category
- Production Cost Analysis by Category

## Dashboard Insights

### Monthly Performance
- Tracks monthly profit and discount trends.
- Helps identify seasonal sales patterns.
- Measures the impact of discounts on profitability.

### Yearly Comparison
- Compares profit and discount between 2019 and 2020.
- Highlights significant business growth.

### Category Analysis
- Identifies the most profitable product categories.
- Compares production costs across categories.
- Supports data-driven business decisions.

## How to Use

1. Open the **Sales Performance.pbix** file using Power BI Desktop.
2. Load or refresh the dataset.
3. Use the slicers to filter data by Category, Month, or Year.
4. Explore KPIs and visualizations to gain business insights.
